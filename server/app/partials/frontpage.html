APPRIS

The APPRIS Database [2] houses protein structure, function and conservation information for splice variants in five genomes. The APPRIS WebServer [1] executes a range of computational methods to annotate splice variants for individual genes. As part of its annotation process APPRIS selects a single CDS as the principal isoform for each gene.

API

The APPRIS Database and APPRIS WebServer can be accessed through a range of RESTful web services. These services can retrieve both gene/transcript information and the results of the individual APPRIS modules.
